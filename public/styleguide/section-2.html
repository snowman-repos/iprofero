2.1 Button

Buttons indicate a possible user action. Buttons may be grouped, contain only icons, icons with text, and may be separated by conditionals. They may also be attached to other content elements.

Button

Button
:active

A button can show it is currently the active user selection

Button
:focus

A button can change when pressed using touch or mouse events

Button
.disabled

A button can show it is currently unable to be interacted with

Button
.loading

A button can show a loading indicator

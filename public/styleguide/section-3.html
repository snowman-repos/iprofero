3.4 Form

A form is a collection of user input elements. Forms always include fields, and fields always contain form elements. Fields themselves may also include: inputs, standard form fields, labels, selection dropdowns, textareas, as well as: checkboxes, and message blocks. Validation messages use messages which are formatted for use inside forms.
